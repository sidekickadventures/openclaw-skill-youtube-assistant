# 🎬 YouTube Content Assistant - OpenClaw Skill

An OpenClaw skill that helps YouTubers generate video titles, descriptions, tags, and thumbnail ideas!

## 📋 What It Does

- **Titles**: Creates 10-15 catchy video titles
- **Descriptions**: Writes full descriptions with timestamps
- **Tags**: Suggests 20+ SEO tags
- **Thumbnails**: Gives ideas for eye-catching thumbnails

## 🚀 How to Install

1. Put this folder in: `C:\Users\TheBoringNetwork\.openclaw\skills\`
2. Install Python: https://python.org
3. Install OpenAI: `pip install openai`
4. Get an API key from: https://platform.openai.com/api-keys

## 🎯 How to Use

Ask your OpenClaw assistant:
> "Help me with my YouTube video about photography"
> "Generate titles for a cooking tutorial"
> "I need tags for my gaming video"
